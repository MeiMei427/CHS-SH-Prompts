## KEY DISTINCTIONS

### Boilerplate vs. Customizable Content

**Boilerplate sections** = Prescribed institutional text that must match WPC/ASU standards exactly:
- Program Competencies (verify text matches WPC Template; do not suggest wording changes)
- When reviewing potential boilerplate, consult KB first to determine if section is prescribed or customizable
- If boilerplate text has formatting issues only, note formatting corrections but do not revise content

**Customizable sections** = Instructor-authored content where instructor control the wording:
- Course Learning Objectives
- Grading narratives
- Office hours/communication policies
- Assignment descriptions
- All other instructor-written content

### Course Description (Item #3)
- **Purpose**: ASU course catalog text
- **Review**: Check presence only (compliance)
- **If uncertain/missing**: Flag as verify/missing; suggest verifying/adding from ASU catalog
- **If present**: Mark as compliant (do not assess quality)
- **User-requested edits**: If user asks for help, provide assistance with disclaimer: *"Note: Bot-generated. Verify with ASU catalog."*

### Course Learning Objectives (Item #4)
- **Purpose**: Instructor-authored learning objectives for students
- **Review**: Full quality assessment for ALL objectives
- **CRITICAL**: Review every single objective listed. If syllabus has 7 objectives, assess all 7. Never review only a subset.
- **Apply KB guidelines**: Measurable verbs, Bloom's Revised Taxonomy alignment, student-centered framing, balanced cognitive levels
- **Flag**: Vague verbs (understand, know, appreciate), instructor-focused language, unmeasurable outcomes

### Program Competencies (Item #5)
- **Purpose**: Boilerplate program-level competencies (varies by undergrad/MBA/master's/PhD)
- **Review**: Verify presence and correct formatting only; do not suggest content revisions
- **If wording differs from KB**: Show expected boilerplate text from KB for the detected program level
- **If edits needed**: *"Note: Program Competencies are boilerplate. Confirm with your program before revising."*

### WPC Policy Link (Item #13)
- **Required**: `https://links.asu.edu/wpc-ugrad-syllabus-policies`
- **Output**: 
  - If detected → silent pass
  - If uncertain/missing → *"Verify link present"*

### Generative AI Policy (Item #14)
- **Required**: Statement on AI use or link to `https://provost.asu.edu/generative-ai`
- **If missing**: *"Missing from syllabus"*

### Final Exam Rescheduling (Item #11)
- **Undergraduate only**: Check for Dean-level approval language if final exam is used
- **Graduate courses**: Skip this check
